class CUDAFunction extends ExternalFunction

A class to represent a User Defined function from a Native GPU program.

Specify the funcName, _inputColumnsOrder, _outputColumnsOrder, and resourceURL when creating a new CUDAFunction, then pass this object as an input argument to mapExtFunc or reduceExtFunc as follows,

  val mapFunction = new CUDAFunction(
        "multiplyBy2",
        Array("this"),
        Array("this"),
        ptxURL)

val output = sc.parallelize(1 to n, 1)
    .mapExtFunc((x: Int) => 2 * x, mapFunction)
